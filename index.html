Explora nuestra página web que te ayudará a aprender sobre los elementos básicos de HTML.

Ejemplos de etiquetas HTML utilizadas

Descripción: mostrar un párrafo usamos la etiqueta: <p>

Ejemplo visual:

Codigo:

<p>Este es un párrafo de ejemplo</p>

Este es un párrafo de ejemplo.

Descripción:Para mostrar un encabezado usamos la etiqueta: <h1>, <h2>, etc.

Ejemplo visual:

Codigo:

<h3>Encabezado de nivel 3</h3>

Encabezado de nivel 3

Descripción:Para mostrar una imagen usamos la etiqueta: <img>

Ejemplo visual:

Codigo:

<img src="https://upload.wikimedia.org/wikipedia/commons/thumb/a/a7/React-icon.svg/512px-React-icon.svg.png" alt="Imagen de prueba" width="300">

Descripción:Para mostrar un video usamos la etiqueta: <video>

Ejemplo visual:

Codigo:

<video src="https://www.w3schools.com/html/mov_bbb.mp4" controls width="300"></video>

Descripción:Para reproducir audio usamos la etiqueta: <audio>

Ejemplo visual:

Codigo:

<audio src="https://files.freemusicarchive.org/storage-freemusicarchive-org/music/no_curator/Komiku/Komiku_-_Its_time_for_adventure/Komiku_-_04_-_Overworld.mp3" controls></audio>

Descripción:Para crear una lista no ordenada usamos <ul> y <li>

Ejemplo visual:

Codigo:

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
  • HTML
  • CSS
  • JavaScript

Descripción:Para crear una tabla usamos <table>, <tr>, <td> y <th>

Ejemplo visual:

Codigo:

<table>
  <tr>
    <th>Etiqueta</th>
    <th>Descripción</th>
  </tr>
  <tr>
    <td>&lt;p&gt;</td>
    <td>Párrafo</td>
  </tr>
  <tr>
    <td>&lt;img&gt;</td>
    <td>Imagen</td>
  </tr>
</table>
Etiqueta Descripción
<p> Párrafo
<img> Imagen

Descripción:Para crear un formulario usamos la etiqueta: <form>

Ejemplo visual:

Codigo:


<form>
  <label>Nombre:</label><br>
  <input type="text" name="nombre"><br><br>
  <label>Mensaje:</label><br>
  <textarea name="mensaje" rows="4" cols="30"></textarea><br><br>
  <input type="submit" value="Enviar">
</form>






Descripción:Para aplicar estilo en línea se usa el atributo style=""

Ejemplo visual:

<p style="color:green;">Este texto es verde gracias al estilo en línea.</p>

Este texto es verde gracias al estilo en línea.
